# 知了二维码

#### 项目介绍
小程序项目 二维码生成器 《微信小程序》

#### 软件架构
软件架构说明 原生微信小程序

## 基础功能
> 二维码生成器，文字、url等生成二维码

> 二维码反向解析器。 二维码反向解析出来字符串

> 微信名片，本地存储的一个微信名片，简单实现
